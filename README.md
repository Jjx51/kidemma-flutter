# KIDEMMA

## Descripción General
Aplicación móvil desarrollada con Flutter.

## ¿Cómo empezar?
Para poder ejecutarla, necesitarás tener Flutter instalado en tu entorno.

### Prerrequisitos

1.  **Instalar Flutter SDK:** Sigue las instrucciones en la [documentación oficial de Flutter](https://flutter.dev/docs/get-started/install) para instalar el SDK.
2.  **Instalar Visual Studio Code:** Descarga e instala [VS Code](https://code.visualstudio.com/).
3.  **Instalar la extensión de Flutter en VS Code:**
    *   Abre VS Code.
    *   Ve a la vista de Extensiones (`Ctrl+Shift+X`).
    *   Busca `Flutter` e instálala. Esta extensión también instalará la extensión de `Dart`.

### Ejecutar la aplicación

1.  Clona este repositorio en tu máquina local.
2.  Abre la carpeta del proyecto en Visual Studio Code.
3.  Abre una terminal integrada en VS Code (`Ctrl+`` ` ``).
4.  Ejecuta el siguiente comando para obtener todas las dependencias del proyecto:

    ```sh
    flutter pub get
    ```
5.  Ejecuta la aplicación con el siguiente comando:

    ```sh
    flutter run
    ```

Esto iniciará la aplicación en un emulador, simulador o dispositivo físico conectado que tengas configurado.
